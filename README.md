## vue3-mallmanagement

1. 项目使用的事`pnpm`的包管理工具，所有命令请使用pnpm
2. 启动项目

```sh
# 安装依赖
$ pnpm  i

# 启动项目
$ pnpm run dev
```

3. 代码提交请使用

```sh
# 方式一 进行提示操作
$ pnpm commit

# 方式二使用git命令进行操作 按照特定的格式进行提交
$ git add .
$ git  commit -m  "[type]: git提交描述信息 "

```

### 项目相关

1. 账号和用户查看`/mock/user.ts`文件
2. 登录账号会看到不同的菜单以及首页上的`按钮`
3. admin 看到的是全量的
